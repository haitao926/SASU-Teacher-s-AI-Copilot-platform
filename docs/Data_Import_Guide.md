# 系统初始化与数据导入操作手册 (Admin 界面版)

本手册指导管理员通过**Web 管理界面**完成学生数据初始化与成绩导入，无需使用命令行工具。

## 0. 准备工作

请确保您拥有以下 Excel 文件（.xlsx 格式）：

1.  **学生总名单 (Master Roster)**
    *   必须包含列：`学号` (或 Student ID), `姓名` (Name)。
    *   可选包含列：`班级` (Class)。
2.  **考试成绩单**
    *   必须包含列：`姓名`。建议包含 `学号` 以防重名。
    *   包含各科成绩（如语文、数学、英语、6总等）。

---

## 1. 第一步：学生名单初始化

**目的**：建立全校学生的电子档案库。

1.  **登录门户**：使用管理员账号登录 `http://localhost:5173`。
2.  **进入管理页面**：
    *   点击侧边栏的 **“教务管理”** -> **“学生档案”**。
3.  **执行导入**：
    *   点击工具栏右侧紫色的 **“批量导入”** 按钮。
    *   选择您的《学生总名单》Excel 文件。
    *   系统会自动解析并显示预览条数。点击“确定”开始导入。
4.  **结果确认**：
    *   导入成功后，列表会自动刷新。
    *   您可以使用顶部的搜索框（输入姓名或学号）验证学生是否已在库中。

> **提示**：如果导入了重复的学生，系统会自动更新其信息（如班级），不会创建重复记录。

---

## 2. 第二步：考试成绩导入

**目的**：录入历次考试的分数。

1.  **进入应用**：
    *   在左侧菜单点击 **“学情洞察”** (或直接点击 **“学生成绩统计”** 应用)。
    *   点击顶部的 **“数据导入”** 按钮。
2.  **填写信息**：
    *   设置考试名称（例如：“2026届高二下期末”）。
    *   选择对应的届数、年级、学期。
3.  **上传文件**：
    *   上传《考试成绩单》Excel 文件。
    *   系统会自动将成绩与刚才建立的学生档案进行匹配。
4.  **查看结果**：
    *   导入完成后，切换到“成绩排名与分析”视图，检查是否有数据。

---

## 3. 数据维护与纠错

### 场景 A：个别学生信息错了（如名字写错、分班错了）
1.  进入 **“教务管理” -> “学生档案”**。
2.  搜索该学生。
3.  点击右侧的 **“铅笔”图标** 进行编辑。
4.  修改后点击保存，即刻生效。

### 场景 B：学生忘记密码
1.  进入 **“教务管理” -> “学生档案”**。
2.  找到学生，点击 **“锁”图标**。
3.  确认后，密码将被重置为其**学号**。

### 场景 C：某次考试导入严重错误（如列对齐错误）
1.  进入 **“学生成绩统计”** 应用。
2.  在顶部下拉框选择那场错误的考试。
3.  点击右侧红色的 **“删除”** 按钮（垃圾桶图标）。
4.  确认删除后，该场考试的所有数据会被清除。
5.  修改 Excel 文件后重新导入。

### 场景 D：彻底删除一名学生
1.  进入 **“教务管理” -> “学生档案”**。
2.  点击 **“红色垃圾桶”** 图标。
3.  **警告**：这将同时删除该学生的所有历史成绩记录和作业记录，且**不可恢复**。

---

通过以上步骤，您可以完全在浏览器中完成对学校教务数据的初始化与日常维护。
