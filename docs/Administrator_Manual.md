# SASU 教学辅助平台 - 管理员操作手册

**版本**：v1.0  
**适用对象**：系统管理员、教务老师  
**更新日期**：2026-01-22

---

## 1. 系统概述

本平台旨在为学校提供一体化的教务数据管理与学情分析服务。作为管理员，您拥有系统的最高权限，负责维护**学生基础档案**和**考试成绩数据**，确保上层应用（如成绩分析、证明打印）的数据准确性。

### 核心管理模块
1.  **教务管理**：管理学生名单、账号与基础信息。
2.  **学情中心**：管理考试场次、录入与维护成绩数据。

## 3. 模块一：学生档案管理 (Student Roster)

这是系统的数据基石。所有成绩、作业、考勤数据都依赖于此处的学生档案。

**功能入口**：左侧菜单栏 -> **教务管理** -> **学生档案**

### 3.1 批量导入学生 (初始化)
新学年开始或系统初始化时使用。

1.  准备 Excel 文件，需包含表头：`学号`, `姓名`, `班级` (可选)。
2.  点击页面右上角的紫色 **“批量导入”** 按钮。
3.  选择文件，系统将自动解析预览。
4.  确认无误后点击导入。系统会自动去重（已存在的学生会更新信息，不会重复创建）。

### 3.2 日常维护
*   **搜索**：在顶部搜索框输入姓名或学号，支持模糊搜索。
*   **筛选**：输入班级号（如 `1`）可只查看该班学生。
*   **编辑信息**：点击列表右侧的 **“铅笔”** 图标，可修正学生姓名或班级。
*   **重置密码**：点击 **“锁”** 图标。当学生忘记密码时使用，密码将重置为其**学号**。
*   **删除学生**：点击 **“红色垃圾桶”** 图标。
    *   **⚠️ 高危操作**：删除学生将**永久清除**该学生的所有历史成绩、作业提交记录和账号信息。仅在学生退学或录入错误时使用。

## 4. 模块二：考试成绩管理 (Exam Data)

管理全校的考试数据资产。

**功能入口**：左侧菜单栏 -> **学情中心** -> **学生成绩统计**

### 4.1 录入新考试成绩
1.  进入应用，点击顶部的 **“数据导入”** 标签。
2.  配置考试信息（届数、年级、学期、考试类型）。
3.  上传成绩 Excel 文件。
    *   系统会自动根据“姓名”匹配档案库中的学生。
4.  点击 **“开始导入”**。

### 4.2 成绩数据纠错
如果发现某次考试的数据有问题（如列对齐错误、分数录入错误）：

*   **方案 A（覆盖更新）**：
    *   在 Excel 中修正数据。
    *   使用**完全相同**的考试名称再次执行导入流程。
    *   系统会自动覆盖旧的分数。

*   **方案 B（整场删除）**：
    *   在页面顶部的考试下拉框中，选中该场错误的考试。
    *   点击右侧红色的 **“删除” (垃圾桶)** 按钮。
    *   确认后，该场考试及其包含的数千条成绩记录将被彻底清除。

### 4.3 数据导出
*   选中某场考试后，点击 **“导出 CSV”** 按钮，可获取原始数据表，用于线下存档或汇报。

## 5. 模块三：成绩证明自助打印

为学生提供服务的功能。

**功能入口**：左侧菜单栏 -> **学情中心** -> **成绩证明生成器** (或独立入口)

1.  **查找学生**：输入姓名找到目标学生。
2.  **选择考试**：在右侧列表中勾选需要展示在成绩单上的考试（支持多选）。
3.  **生成文件**：点击 **“生成成绩证明”**，系统将自动排版并生成 PDF 文件供打印。

## 6. 常见问题处理 (Troubleshooting)

| 问题现象 | 可能原因 | 处理建议 |
| :--- | :--- | :--- |
| **导入成绩时提示“找不到学生”** | 该学生尚未在“学生档案”中建立资料 | 请先去“教务管理”中添加该学生，或检查 Excel 中名字是否有错别字。 |
| **导入后某科目成绩显示为 0 或 -** | Excel 中该单元格为空或格式错误 | 检查 Excel，确保分数为数字格式，然后重新导入覆盖。 |
| **学生无法登录** | 密码错误或账号被禁用 | 在“学生档案”中找到该学生，点击“重置密码”。 |
| **成绩单上的排名不准** | 导入时未包含排名列 | 请确保上传的 Excel 中包含`6排`、`班排`等列，或联系技术支持重新计算。 |
