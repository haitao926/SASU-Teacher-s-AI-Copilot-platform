## 配置说明

### 添加新应用入口
编辑 `public/config/entries.json`，添加新的应用条目：

```json
{
  "id": "your-app-id",
  "name": "应用名称",
  "description": "应用描述",
  "icon": "gradient-purple",
  "tags": ["标签1", "标签2"],
  "url": "https://your-app.com?source=iai-portal",
  "status": "available",
  "featured": true,
  "group": "assessment",
  "usage": 0,
  "order": 10
}
```

### 添加公告
编辑 `public/config/announcements.json`，添加新公告：

```json
{
  "id": "ann-xxx",
  "title": "公告标题",
  "content": "公告内容",
  "time": "2025-12-17 17:00",
  "tag": "新功能",
  "tagType": "info",
  "pinned": true,
  "read": false
}
```
